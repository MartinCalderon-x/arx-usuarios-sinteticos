Roadmap de Desarrollo

Plan de implementacion por fases

Fase 1: Setup e Infraestructura

  • Inicializar monorepo (backend + frontend)
  • Configurar FastAPI con estructura base
  • Configurar Vite + React con routing
  • Setup Supabase (DB + Auth + Storage)
  • Configurar Cloud Run para despliegue

Fase 2: Integracion Vision AI

  • Configurar API de Gemini Vision
  • Implementar placeholder para Qwen 2.5 VL
  • Crear servicio de analisis de imagenes
  • Implementar generacion de heatmaps
  • Focus maps y AOI

Fase 3: Sistema de Arquetipos

  • Modelo de datos para usuarios sinteticos
  • API CRUD completa
  • Templates base (consumidor, operario, etc.)
  • Embeddings con pgvector
  • Busqueda semantica de arquetipos

Fase 4: Interaccion con LLM

  • Prompt engineering para simulacion
  • Contextualizacion con arquetipos
  • Evaluacion de prototipos
  • Chat interactivo con usuarios sinteticos

Fase 5: Expansion

  • Navegacion de flujos multi-pantalla
  • Integracion con Figma (plugin)
  • Reportes avanzados y exportacion
  • Dashboard de metricas consolidado
